hexterm
=======

Convert hexadecimal color codes to Xterm

## Install

Clone, download and extract or install with npm

```sh
npm install hexterm
```

## Usage

```js
const hexterm = require('hexterm')

hexterm('005fd7') // 26
hexterm('#005fd7') // 26
hexterm('005ED9') // 26
hexterm('ff0') // 11
hexterm('#ff0') // 11
```
